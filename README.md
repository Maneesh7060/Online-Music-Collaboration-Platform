# Online Music Collaboration Platform

A Java-based web application using JSP, Servlets, JDBC, and MySQL.

## Features
- User Registration & Login
- Upload and manage music files
- Real-time collaboration using Firebase
- File versioning for uploaded music
- Secure and validated user inputs

## Technologies Used
- JSP, Servlets
- JDBC (MySQL)
- Firebase Realtime Database
- HTML, CSS
- Apache Tomcat

## Review 2 Enhancements
- Implemented real-time collaboration using Firebase
- Added client-side and server-side validation
- Improved error handling and robustness
- Introduced file versioning for media uploads
- Modular MVC-based architecture

## How to Run
1. Import project into Eclipse / VS Code
2. Create database using `database.sql`
3. Update DB credentials in `DBConnection.java`
4. Deploy on Apache Tomcat Server
